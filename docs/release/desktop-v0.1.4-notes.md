# MoneySiren v0.1.4

> Distribution status: this version was tagged, but no GitHub Release or
> versioned runtime assets were published.

MoneySiren v0.1.4 focused on open-source adoption documentation, diagnostics,
and a credential-free review path.

## Added

- Provider support and permission matrices.
- Demo scenarios and troubleshooting guidance.
- Good-first-issue guidance.
- Question and private-security entry points.
- Improved doctor and connection diagnostics.

## Known limitation

The isolated Windows source-free install smoke was blocked because no matching
v0.1.4 GitHub Release runtime was published.

## Security

- Provider connectors remained read-only.
- Telemetry remained disabled by default.
- Public examples and screenshots were required to use fake data.
